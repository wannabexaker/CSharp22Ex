# FinalBossEnterprize_Support

This folder contains the supporting layers for the FinalBossEnterprize sample:

- `Services/` — logging and other cross-cutting services
- `Views/` — Razor-style console rendering
- `Utilities/` — helper extensions

Combine this with `FinalBossEnterprize_Core` in a single project to get the full application.
Namespace used: `FinalBossEnterprize`.
